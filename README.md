# Embutido-King Landing Page

Simple, single-page landing site for Embutido-King Filipino catering business.

## Features

- Clean, mobile-responsive design
- Menu organized by category (Specialty, Noodles, Pork, Chicken, Beef, Goat, Vegetables)
- Click-to-call phone number
- Warm color scheme (deep red, gold, cream)

## Tech Stack

- Pure HTML/CSS (no frameworks)
- Single file for easy deployment
- GitHub Pages ready

## Local Development

Run a local server:

```bash
python -m http.server 8000
```

Then open: http://localhost:8000

## Deployment

This site is designed to be deployed on GitHub Pages.

### Quick Deploy:

1. Push to GitHub
2. Go to repo Settings > Pages
3. Set source to "main" branch, root folder
4. Site will be live at: `https://yourusername.github.io/embutido-king/`

### Custom Domain:

To use embutido-king.com:
1. Add CNAME file with domain name
2. Configure DNS at domain registrar:
   - Add A records pointing to GitHub Pages IPs
   - Or add CNAME record pointing to yourusername.github.io
